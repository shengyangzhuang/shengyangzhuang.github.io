Software design for state-of-the-art volumetric 3D printer

Current volumetric 3D printing algorithms offer a wide range of options. Some advanced algorithms, such as computed axial lithography, penalty minimization, and object space model optimization, have been developed. However, these algorithms have not been combined or integrated into a user-friendly application to enhance the volumetric printing experience.

This project aims to create an easy-to-use volumetric printing interface using Python scripts from the vamtoolbox developed by the UC Berkeley group. The resulting app includes essential features for volumetric printing, including a light calibration function, an integrated motor control section, a variety of optimization algorithms, and detailed settings for projection and optimization functions.

SonoPrint: Acoustically-Assisted Volumetric 3D Printing

Advancements in additive manufacturing have revolutionized fields like aerospace and medical devices by allowing precise control over material properties. However, current technology has limitations, such as restricted reinforcement strategies.

To address current limitations, we developed SonoPrint, an acoustically-assisted volumetric 3D printer. By creating a standing wave field that patterns microparticles within structures, it offers unprecedented control over material properties.

Implementation of the Pre-defined Behaviors of Differential-drive Robot (KTH course project)

This project aimed to control a differential-drive robot to follow a predefined behavior in a workspace. We analyzed the robot's kinematics, abstracted its motion into a Transition System, and created a high-level plan. To achieve this, we developed a hybrid controller with a rotation and go-to-line component. A practical application involved a hospital robot delivering medicine, navigating corridors by rotating to face its destination and moving in straight lines until it reached its goal.

Zebrafish object detection for intelligent micromanipulation robotics systems (HIT course project)

Intelligent Micromanipulation Robotics Systems focus on precise microinjections to potentially treat cancer patients. The process involves extracting cancer cells, cultivating them, and then performing micro manipulations to identify specific medicines for treatment. Zebrafish were used as a model to test different system functions. In this project, I learnt how to develop an algorithm to achieve object detection.
